# My Personal Portfolio 

Hey there Welcome to Profile Page repository. I built this site to showcase my journey as a Full Stack Developer and share a bit about who I am and what I do.

## What's Inside? 

- **About Me**: A little introduction about myself and my passion for coding
- **Technical Skills**: Tools and technologies I work with
- **Projects**: Some cool stuff I've built
- **Experience**: Where I've worked and what I've done
- **Education**: My academic background
- **Contact**: How to get in touch with me

## Built With 

This portfolio is intentionally kept simple and lightweight, built using:
- HTML5
- CSS3 


## Running Locally 

1. Clone this repo
   ```bash
   git clone https://github.com/PrafulShinde45/Profile-Page.git
   ```
2. Open `index.html` in your browser
3. That's it No build process, no dependencies
